# Gandul's Calculator

Aplikasi kalkulator sederhana dan bergaya yang dibuat dengan HTML, JavaScript, dan Tailwind CSS.

## Fitur

- **Aritmatika Dasar**: Melakukan penjumlahan, pengurangan, perkalian, dan pembagian.
- **Antarmuka Bersih**: Desain yang ramah pengguna dengan tampilan yang jelas dan tombol besar yang mudah ditekan.
- **Desain Responsif**: Didisain menggunakan Tailwind CSS untuk tampilan dan nuansa modern.
- **Manajemen Input**:
  - Tombol `AC` untuk menghapus seluruh entri.
  - Tombol Backspace (ikon hapus) untuk menghapus karakter terakhir.

## Teknologi yang Digunakan

- **HTML5**: Untuk fondasi struktural aplikasi.
- **Tailwind CSS**: Untuk gaya utility-first dan tata letak responsif.
- **JavaScript**: Untuk logika kalkulator dan interaktivitas.

## Cara Memulai

1. Clone repositori:
   ```bash
   git clone https://github.com/Arga-12/calculator.git
   ```
2. Buka `index.html` di browser web Anda.
3. Mulai menghitung!
