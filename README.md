# Gandul's Calculator

A simple, stylish calculator application built with HTML, JavaScript, and Tailwind CSS.

## Features

- **Basic Arithmetic**: Perform addition, subtraction, multiplication, and division.
- **Clean Interface**: User-friendly design with a clear display and large, easy-to-press buttons.
- **Responsive Design**: Styled using Tailwind CSS for a modern look and feel.
- **Input Management**:
  - `AC` button to clear the entire entry.
  - Backspace button to remove the last character.

## Technologies Used

- **HTML5**: For the structural foundation of the application.
- **Tailwind CSS**: For utility-first styling and responsive layout.
- **JavaScript**: For the calculator logic and interactivity.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Arga-12/calculator.git
   ```
2. Open `index.html` in your web browser.
3. Start calculating!
